# embed-wider-flickr-photo
WordPress plugin to make oembed-embedded Flickr photo wider

# requirements

  * PHP5.5 or above
  * WordPress 4.4.1 or above
  * phpunit 5.1.6 or above (test requires PHP5.6 or above)
